## Auto-redirecting

當使用者訪問到某個網站時，自動把使用者轉向其他網頁或網站的一種技術。

合理的使用方法包括了：把用戶轉到指定的網頁版本；域名變更或刪除轉到新網域，等。如果不合理使用的話，搜索引擎會認為這是對使用者的誤導，而對這種網站施以懲戒。
