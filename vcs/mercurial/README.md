# Mercurial

[Mercurial](https://www.mercurial-scm.org/) 是一套版控軟體，主要使用 Python 撰寫。[Bitbucket](../bitbucket.md) 有提供托管服務。

## 安裝

### Mac

    brew install hg

## clone 專案

    hg clone [URI]
