# Software Metric

[Software Metric][] 是用科學客觀的方法來衡量程式碼，其產生的結果對於程式碼品質或是偵錯規劃都是有幫助的

五種指標

* Lines of Code 程式碼行數
* Maintainbility Index 可維護性指標
* Cyclomatic Complexity 循環複雜度
* Depth of Inheritance 繼承深度
* Class Coupling 類別結合程度
