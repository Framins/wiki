# AWS CLI

使用方法詳細可以參考 [官方網頁](https://aws.amazon.com/tw/cli/) 或 [GitHub 專案](https://github.com/aws/aws-cli)

## Installation

### Linux

使用 pip 安裝

    pip install awscli

### MAC

使用 brew 安裝

    brew install awscli
