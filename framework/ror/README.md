# [Ruby on Rails](http://rubyonrails.org/)

`Ruby` `Full-stack Framework`

簡稱為 `RoR`，或是 `Rails`。是一個建構在 Ruby 上的 MVC 框架。

官方網站上有寫版本的對應

* Ruby 1.8.x 只能對應到 Rails 3.2 以前
* Ruby 1.9.2+ 可以對應到 Rails 5
