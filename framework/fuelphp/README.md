FuelPHP
=======

`Full-stack Framework`

[FuelPHP](http://fuelphp.com/)

Getting Started
---------------

先下載官方的指令

```bash
curl get.fuelphp.com/oil | sh
```

然後就可以用 `oil` 指令了

```bash
oil create project
```

它所會做的事是使用 git clone 下載原始碼，然後使用 composer 安裝相依套件，蠻棒的懶人包
